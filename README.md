# TicTacToe in Python/Tkinter

It is exactly what it sounds like.

## Requirements

Linux:
- python-tkinter
- python 3.7.6

## Usage

git clone https://github.com/plarabee/tictactoe.git
cd tictactoe
python3 tictactoe.py
